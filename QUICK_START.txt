The chatbot now UNDERSTANDS natural requests WITHOUT commands!

Instead of:   "add task finish homework"

More examples:

  ✅ "remind me to study at 15:30" → Sets reminder
  ✅ "im working on homework"      → Starts tracking
  ✅ "what is 5 times 10"          → Calculates
  ✅ "what does code mean"         → Defines word
  ✅ "schedule meeting on 2024-03-25" → Adds event

🌐 WEB VERSION (Recommended - Share with Friends!)
═══════════════════════════════════════════════════════════

1. Double-click: run_web.bat
   OR
   Command: python app.py

2. Wait 2-3 seconds, then open:
   http://localhost:5000

3. Start chatting! Type your message and press Enter

4. Share with friends:
   - Get your IP: Open PowerShell, type: ipconfig
   - Copy "IPv4 Address" (e.g., 192.168.1.100)
   - Send them: http://[YOUR-IP]:5000
   - They must be on same WiFi network

❓ TROUBLESHOOTING
═══════════════════════════════════════════════════════════

Port 5000 in use?
  → Close other apps using port 5000
  → Or run: python app.py --port 5001

Can't access from another device?
  → Make sure both on same WiFi
  → Check firewall allows port 5000
  → Try the IP address instead of localhost

Request not understood?
  → Try natural phrasing: "i need to..."
  → Include dates/times when needed
  → See NLP_GUIDE.md for examples

Any errors?
  → Make sure Flask is installed: pip install flask
  → Check Python 3.12 is being used
